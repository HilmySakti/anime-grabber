# What the minus side ?
- No Link Cache 

# How to use ?
- Clone this git
- Create Virtual Enviroment or without Virtual Enviroment
- Run command:
```bash
pip install -r requirement.txt
```
- Goto directory that you have clonned
- Run command:
```bash
python -m gui
```

- Select any anime, and you get the download link
